How To Copy

Follow these steps exactly to get your PowerShell code:

  1. Go to the Roblox game you want to copy in your browser.
  2. Press F12 to open Developer Tools.
  3. Click on the Network tab at the top.
  4. Press F5 or click refresh on the game page.
  5. Look at the file list. Click the FIRST file at the top.
  6. Right-click that file, hover over Copy.
  7. Select Copy as PowerShell from the menu.
  8. Paste the code here and click "COPY GAME NOW".
